# cbc-calculators-and-aimer
REMEMBER THIS IS AN VERY EXPERIMENTAL CALCULATORS AND AIMER FOR CREATE BIG CANNON'S
you can change the data in "cannondata.txt"

the structure is like this :
- the first 3 lines is cannon x, y and z
- the second 3 lines is target x, y and z


- Create_SequencedGearshift_0 <= For Controlling Pitch
- Create_SequencedGearshift_1 <= For Controlling Yaw


- n <= directions (north)
